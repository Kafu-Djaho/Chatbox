Step 1 - Add Node.js ppa
The Node.js package is available in the LTS release and the current release. Itâ€™s up to the user to select the version of Node that is required. Run the following command to add the PPA to the Ubuntu system:

sudo apt-get install curl
curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -

Step 2 - Install Node.js on Ubuntu
After successfully adding the PPA to the system, execute the command below to install Node on Ubuntu:

sudo apt-get install nodejs

Step 3 - Verification
To verify any installation, check the version number of the installed software. For node, run the following command:

node -v
